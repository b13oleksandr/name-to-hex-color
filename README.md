# Install
```
npm i name-to-hex-color
```

```
yarn add name-to-hex-color
```

# Usage
``` js
import { nameToHexColor } from 'name-to-hex-color'

const color = nameToHexColor('Bender Bending Rodríguez')
```

In Node.js:
``` js
const { nameToHexColor } = require('name-to-hex-color')

const color = nameToHexColor('Bender Bending Rodríguez')
```
